# sut-ec-mobile テストシナリオ一覧

> **第2段階での差分**: 本一覧は第1段階（モック）前提。実サーバー接続版では認証・初期データ・注文金額などが変わる。[テスト計画書 §0.1](test-plan.md#01-第2段階サーバー実装でのテストへの影響) を各シナリオに上書き適用すること。

各シナリオ(SC-xx)は [テスト観点](test-viewpoints.md) を実施単位に落としたもの。詳細な操作手順・期待結果は [テスト手順書](test-procedures.md) の対応 TC を参照。両 OS(Android / iOS)で実施する。

- **優先度**: 高 / 中 / 低([テスト計画書 6.1](test-plan.md#61-テスト優先度))
- **種別**: 機能 / フロー / 同期 / i18n / 境界 / 異常 / 非機能
